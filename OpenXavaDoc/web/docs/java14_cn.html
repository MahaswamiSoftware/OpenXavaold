Java 1.4

OpenXava默认兼容Java 5，你可以按照以下步骤在Eclipse中重新编译它，使得它能够兼容Java 1.4：
  • Windows > Preferences > Java > Installed JREs : 选择Java 1.4
  • OpenXava > Properties > Java Compiler > Compiler compliance level: 选择1.4
  • OpenXava > Properties > Java Builde Path > Source > 移除 java5-src 文件夹
  • Build all (Ctrl -B)全部编译
如果你希望使用POJOs + JPA + Java 5 annotations 你必须使用Java 5。Java 1.4 只能使用标准XML组件。

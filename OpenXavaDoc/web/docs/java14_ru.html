Java 1.4

Несмотря на то, что OpenXava поставляется компилированной для Java 5, она может быть без особых проблем пересобрана с использованием Java 1.4.
Далее рассказано, как это сделать в Eclipse:
  • Windows > Preferences > Java > Installed JREs : Выберите Java 1.4 среди установленных JRE.
  • OpenXava > Properties > Java Compiler > Compiler compliance level: Проверьте, что уровень совместимости компилятора установлен в 1.4
  • OpenXava > Properties > Java Builde Path > Source > Удалить папку java5-src
  • Запустите полную сборку (Ctrl -B)
К сожалению, вы не сможете использовать аннотации с Java 1.4. Java 1.4 работает только с классическими XML компонентами.

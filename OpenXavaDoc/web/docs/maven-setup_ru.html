Использование Maven

(Также вы можете найти Пример Maven'изированного приложения OpenXava на google code)
Как вы, наверное, уже знаете, OpenXava интенсивно использует сборочные скрипты Ant. Мы расскажем, как можно использовать Maven для построения портлетов OpenXava для использования в Java Portal'ах. Ниже описанная конфигурация оттестирована с релизом OpenXava 4m2.

Сначала необходимо скачать свежий дистрибутив maven 2. Затем создайте pom.xml в папке, где расположен build.xml вшего проекта. Например, для проекта MySchool это будет папка workspace/MySchool. Пример содержимого файла pom.xml:

Затем из командной строки введите команду:
mvn clean install
В результате будет создана папка target. в данной папке будет размещен файл MySchool.war. Скопируйте даннй файл в папку tomcat/webapps/ и запустите tomcat. Теперь можно посмотреть, как работает веб приложение MySchool. Не забудьте, что для корректной работы приложения MySchool необходимо настроить источник данных JNDI и сконфигурировать БД (подробная информация об этом дана в учебнике).

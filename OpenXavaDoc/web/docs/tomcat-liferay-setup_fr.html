Configuration de votre propre Tomcat/Liferay pour des applications OpenXava


Il y a quelques étapes à réaliser avant de pouvoir faire tourner une application OpenXava sur un serveur Tomcat/Liferay.
  • Tout d'abord, il faut installer Liferay.
  • Ensuite, copiez les fichiers suivants dans le dossier {Liferay}/common/lib
    • {OpenXava}/tomcat/common/lib/ejb.jar
    • {OpenXava}/tomcat/common/lib/hsqldb.jar
    • {OpenXava}/tomcat/common/lib/jta.jar
  • Copiez les fichiers suivants dans le dossier {Liferay}/data
    • {OX}/tomcat/data/openxava-db.script
    • {OX}/tomcat/data/openxava-db.properties
  • Editez le fichier {Liferay}/conf/context.xml en ajoutant l'entrée suivante :
    • <Resource name="jdbc/OpenXavaDS" auth="Container" type="javax.sql.DataSource"
    maxActive="20" maxIdle="5" maxWait="10000"
    username="sa" password="" driverClassName="org.hsqldb.jdbcDriver"
    url="jdbc:hsqldb:file:../data/openxava-db"/>
    • Dans ce fichier, vous devez aussi y placer les entrées particulières de votre application
Une fois ces étapes réalisées, vous applications devraient s'exécuter sans problème.

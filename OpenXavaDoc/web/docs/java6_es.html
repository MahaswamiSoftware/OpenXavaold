A partir de v6.1 OpenXava requiere Java 8 o superior

Java 6 (hasta v6.0.2)

Por defecto, OpenXava está compilado con Java 6, lo que te permite usar OpenXava con WebSphere 8 y WebLogic 10. Sin embargo, el servidor de aplicaciones incluido en la distribución de OpenXava es Tomcat 9 (desde OpenXava 6) que requiere Java 8. Por lo tanto, necesitas usar Java 8 (o superior) para seguir la documentación de introducción a OpenXava.
Si quieres usar Java 6 con OpenXava sigue los siguientes pasos:
  • Instala un JDK 6 en tu ordenador.
  • En Eclipse escógelo: Windows > Preferences > Java > Installed JREs : Escoge el JDK 6
  • Descarga e instala Tomcat 7: https://tomcat.apache.org/download-70.cgi
  • Configura tu Eclipse para usar el Tomcat 7 que has descargado en lugar del que incluye OpenXava.

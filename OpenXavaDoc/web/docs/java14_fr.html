Java 1.4


Par défaut, OpenXava est distribué compilé avec Java 5. Toutefois, vous pouvez recompiler OpenXava avec Java 1.4 en suivant les étapes suivantes dans Eclipse :
  • Windows > Preferences > Java > Installed JREs : Choisir une installation Java 1.4
  • Dans le menu contextuel du projet OpenXava : Properties > Java Compiler > Compiler compliance level : Choisir 1.4
  • Dans le menu contextuel du projet OpenXava : Properties > Java Build Path > Source : Supprimer le dossier java5-src
  • Build all (Ctrl-B)
Toutefois, si vous souhaitez utiliser la combinaison POJOs + JPA + Annotations, vous devez utiliser Java 5. Java 1.4 n'est utilisable qu'uniquement avec les composants XML classiques.

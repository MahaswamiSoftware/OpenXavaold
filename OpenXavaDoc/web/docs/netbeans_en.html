Netbeans and OpenXava


This is draft to write about "integration" of OpenXava and Netbeans IDE.

Test Enviroment.
Ox 4.9
Netbeans 8 Beta.

First, import OX folder.

- File / Import Project / Eclipse Project
- Choose OX Workspace folder.
- Check Openxava and OpenxavaTest folders

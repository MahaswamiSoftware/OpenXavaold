Сервер БД HSQL

Для работы с данным руководством вам потребуются базовые знания по работе с базами данных. Также вам необходимо предварительно установить OpenXava и HSQL.

Запуск сервера HSQL

Мы будем использовать базу данных management-db (если она не существует, HSQL создает её автоматически ). Для этого перейдем в папку openxavaX/portal/bin (openxavaX - место, куда мы установили Openxava) и из командной строки выполним команду:
  • В Linux/Unix: ./starthsqldb.sh management-db 1666
  • В Windows: starthsqldb management-db 1666
Мы запустили сервер баз данных HSQL, используя порт 1666.

Администрирование HSQL

HSQL предоставляет утилиту HSQL Database Manager для администрирования базы данных. Скачайте её с web-сайта HSQL и установите. Далее запустите runManagerSwing.bat из папки demo. Теперь мы можем администрировать нашу базу данных management-db, используя удобный графический пользовательский интерфейс. На первой иллюстрации ниже представлено окно подключения к базе данных.
